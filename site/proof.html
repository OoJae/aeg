01

What we learned changed what we built.

The truth-check didn't just confirm the docs — it surfaced behavior that reshaped the design. Three findings, and the decisions they forced.

We found

Recall is global.

In the embedded config, recall(datasets=[x]) doesn't isolate the search — content from another dataset came back anyway.

So we

Never trust a filter for safety.

Quarantine became "never cognify / forget," not filtering; multi-user is honest namespacing, not claimed isolation.

We found

Filtering leaks.

Node-set filtering held in isolation but leaked through shared-entity graph traversal once the graph got busy.

So we

Made exclusion airtight by construction.

Blocked content is never written to the recall substrate — you can't leak what was never stored.

We found

Only deletion is airtight.

Item-level forget() surgically removes a memory in a fifth of a second, with zero LLM calls, and shared entities survive.

So we

Built the immune response on it.

Reversible quarantine and permanent removal both ride forget() — verified before it became the spine.
